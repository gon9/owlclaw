# Reference Slides

スライド生成のクオリティを評価するためのベンチマーク用画像。
プロンプトやテンプレートを改修した際は、これらの参考スライドと比較して品質が劣化していないか確認する。

## ファイル

### `seg_openrouter_exhibit.png`

- **生成日**: 2025-05-28
- **対象記事**: OpenRouter 関連ニュース
- **テンプレート**: `exhibit` (HTML + Puppeteer)
- **採用理由**: 現状の自動生成スライドの中で最もインフォグラフィックとしての完成度が高い参考例
- **目標**: 実運用でも常にこのレベルのスライドが出るようプロンプト・テンプレートを改善する

## 使い方

新しい slide deck の出力を見たときに、ここの PNG と並べて以下を比較する：

1. 情報密度（カード数 / 表 / 図解の有無）
2. レイアウトのバランス
3. 「記事を一目で理解できる」感の強さ

劣化を確認したら `prompts/video-script.md` または `templates/exhibit.html.j2` を見直す。
